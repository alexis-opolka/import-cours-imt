## Linux Shell

```bash
### Si "nom" n'est pas défini, affiche le résultat de la commande "whoami"
echo $(nom-$(whoami))

### Si "nom" n'est pas défini, définir la variable à la valeur "bigboss"
echo ${nom=bigboss}

### Si "nom" n'est pas défini, affiche l'erreur "pas de nom" et sort avec le code d'erreur 1
echo ${nom?"pas de nom"}
```
